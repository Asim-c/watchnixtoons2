# WatchNixtoons2

A Kodi video add-on for streaming cartoons and anime from [watchcartoononline.io](https://www.watchcartoononline.io).   
Help the WatchCartoonOnline website by making a donation to their PayPal at: _wcohelp at yandex dot com_ (verify the address [on this page](http://web.archive.org/web/20190331005116/https://www.watchcartoononline.io/contact)).  

Install it from [this zip here](https://github.com/doko-desuka/plugin.video.watchnixtoons2/raw/master/plugin.video.watchnixtoons2-0.3.1.zip) or from [my repository](https://github.com/doko-desuka/doko.repository/releases) to get automatic updates.  
Pull-requests are always welcome.
